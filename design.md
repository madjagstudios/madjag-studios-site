# MadJag Studios — Design System

## Brand Rule
**Always write "MadJag" with capital M and J only.** Never ALL CAPS, never all lowercase, never "Madjag".

## Color Palette
- Background primary: `#0f0f0f`
- Background secondary: `#161616`
- Background card: `#1c1c1c`
- Red accent: `#9b1c1c` (muted blood red)
- Red accent bright: `#c0392b`
- Red accent glow: `#7f1d1d`
- Text primary: `#f0f0f0`
- Text secondary: `#a0a0a0`
- Text muted: `#555555`
- Border subtle: `#2a2a2a`
- Border accent: `#9b1c1c`

## Typography
- Display / headings: `Bebas Neue` (Google Fonts) — all caps, wide tracking
- Subheadings: `Rajdhani` (Google Fonts) — bold, clean, gamer without being cheesy
- Body: `Inter` — exception here for readability on body text only
- Code / labels: `JetBrains Mono`

## Layout Principles
- Sharp edges everywhere — no rounded corners (or max 2px)
- Asymmetric layouts — hero text left-aligned, accents bleed
- Red accent lines as visual separators (1–2px solid lines)
- Generous negative space in sections
- Cards: dark bg, sharp border, red left-border accent on hover
- Section dividers: angled clip-path cuts or hard horizontal red lines

## Motion
- Page load: staggered fade-up reveals per section (CSS animations, 0.1s delay increments)
- Hero: subtle animated grain/noise overlay
- Cards: translate-y + border-color transition on hover
- No bouncy animations, no playful easing — everything is tight and sharp

## Aesthetic Reference
Modern game studio landing (think Riot, Naughty Dog) but solo dev scale. Dark, confident, a little menacing. Dry humor in copy. Not retro, not pixel art, not neon cyberpunk — just sharp, modern dark.

## Logo
- Geometric "MJ" monogram — angular, blocky
- Red accent element
- Used in nav and hero

## Status Badges
- Live / Released: muted green `#16a34a` bg tint, "LIVE"
- In Progress: amber `#d97706` bg tint, "IN DEV"
- Backlog: muted gray, "BACKLOG"
